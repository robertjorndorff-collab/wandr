Capabilities

Everything you need. Nothing you don't.

📡

Live Streaming

Agent output flows to your channel in real time. TUI noise, spinners, ANSI artifacts — all stripped. Just clean, readable work.

⚡

Command Dispatch

Send prompts to any agent from your phone. !backend sketch the new auth flow — from a trail, a park bench, anywhere.

🟢

Checkpoints

Automatic status: received, working, complete, stalled. Know every agent's state without asking.

📋

Task Queue

Multiple commands queue automatically. Next task fires when the agent goes idle. No dropped work.

🔍

Watchdog

Detects stalled agents, alerts you, auto-restarts on context exhaustion. Nothing goes dark.

🧹

Noise Filtering

Thinking spam, spinner glyphs, duplicate lines — throttled or killed. Your channel stays clean.

🛡️
Trust & Security

Nothing sensitive gets out.

Your agents run locally. Your source code never leaves your machine. But agent output does flow through your messaging platform — that's the whole point. So Wandr stands guard at the gate. Every line passes through a redaction layer before transmission: API keys, tokens, environment variables, connection strings, credentials — pattern-matched and scrubbed. What reaches your channel is clean operational output: status updates, commit messages, work logs. The raw material stays home. The secrets stay buried. Nothing sensitive shall pass.

Open Scaffold

This is the MVP. Make it yours.

Wandr ships today with Claude Code and Slack — because that's what we run and that's what works. But the pattern is the product, not the dependencies. Log tailer, input bridge, checkpoint protocol, task queue — all transport-agnostic. Fork the repo. Swap Slack for Discord, Teams, or a raw websocket. Replace Claude with any AI coding agent that takes a prompt and returns output. This is an open scaffold, not a locked ecosystem. Build your own war room.
